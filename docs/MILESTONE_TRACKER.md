### Lessons Learned

**What Doesn't Work:**
- FSDP with PEFT/LoRA (gradient issues)
- DeepSpeed ZeRO with TRL (incompatible)
- torchrun DDP with LoRA (requires_grad errors)
- accelerate launch with LoRA (gradient issues)

**What Works:**
```python
# The ONLY reliable configuration
model = AutoModelForCausalLM.from_pretrained(
    model_name,
    torch_dtype=torch.bfloat16,
    device_map="auto",  # CRITICAL - model parallelism
    trust_remote_code=True,
)
BATCH_SIZE = 32  # Maximize throughput
# NO accelerate launch, NO DeepSpeed, NO FSDP
```
